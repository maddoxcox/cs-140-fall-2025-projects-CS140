CSS

inline CSS
Inline CSS is CSS that is used by attributes so each <tag> has a CSS attribute.
embedded CSS
Embedded CSS is when a <style> is placed in the <head> of the page.
external CSS
External CSS is CSS that is used by linking another page with CSS.

We only use external CSS so that it is not interfered with by the HTML. There is a heirarchy and external CSS is at the bottom. it also organizes things well.
